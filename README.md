## **Installation & Setup**
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ElpisHR.git
   ```
2. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
3. Start the API server:
   ```bash
   uvicorn main:app --reload
   ```
4. Run the frontend:
   ```bash
   cd frontend
   flutter run
   ```
